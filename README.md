# Requirements (v1)
- User should be able to sign up and sign in
- User should be able to see other posts on it's home feed with likes, comments and caption.
- User should be able to post photos with caption.

# APIs
- /user
    - [POST] /sign-up
    - [POST] /sign-in
    - [POST] /post
- /home
    - [GET] feed
    - [POST] /like/post-id
    - [POST] comment/post-id
